The SaaS onboarding email checklist

The eight lifecycle emails every trial funnel needs, with the trigger and the job each one does.

Start free See how it works
Which emails does a SaaS trial funnel need?

Eight: welcome on signup, a feature nudge when the key feature is unused, a value email, a check in, an upgrade offer when engagement is high, an urgency email after a pricing visit, a win back after five quiet days, and a limit upgrade when a usage cap is hit. Each fires on a behavior, not a fixed day.

If you only build one part of your trial system, build these eight emails. Each one has a trigger and a single job. Tick the box when yours is live and firing on the right condition.

EmailTriggerThe job
1. WelcomeSignup, instantSet one clear first step
2. Feature nudgeKey feature unused in 24hPoint to the activation action
3. Value demoActive, not convertedShow a concrete outcome
4. Check inEngagement droppingInvite a reply, surface objections
5. Upgrade offerScore crosses 70Ask for the upgrade at peak intent
6. UrgencyPricing visit, no convertOne honest push
7. Win backFive days inactiveRe-engage with specific value
8. Limit upgradeUsage limit reachedRemove the blocker

Ready made copy for all eight is in the lifecycle email templates, and the event mapping is in the setup guide. ConversionCRM ships all eight prewired, so starting free turns this checklist on in one step.
