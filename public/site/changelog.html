Product updates

What we have shipped

ConversionCRM is in active development. New features, improvements, and fixes ship weekly during beta. Questions about the roadmap: support@conversioncrm.co.

June 2026
Resources section, blog infographics, nav dropdown
NewImproved
  • Added full Resources library: whitepaper, 2 guides, 2 checklists, 2 templates, case studies hub, and contribute page.
  • Added inline SVG infographics to all 5 blog posts at the most useful anchor points.
  • Fixed Resources nav dropdown to show direct links to each resource type instead of anchor links.
  • Fixed homepage canonical from non-www to www — resolved "Page with redirect" in Google Search Console.
  • Added FAQ page schema (FAQPage JSON-LD) and SoftwareApplication schema to pricing page.
  • Added Q/A direct-answer blocks to all 12 compare and solutions pages for AI engine indexing.
  • Updated sitemap.xml to 36 URLs including all resources, blog, and legal pages.
  • Added llms.txt for AI crawler indexing.
May 2026
Blog launch, lifecycle email engine, dual-path SMTP
NewImproved
  • Launched the blog with 5 articles: conversion benchmarks, 6 levers, freemium vs free trial, PLG vs sales-led, and a pillar guide.
  • Added dual-path email delivery: Resend (default) or customer SMTP. Supports Gmail, Outlook, SES, Postmark, SendGrid.
  • All 8 lifecycle emails now send under the workspace owner's sender name with custom reply-to.
  • Added per-trigger cooldowns and the one-email-per-user-per-batch guardrail.
  • Paying users (Paid stage) are permanently excluded from all lifecycle email.
  • Added the limit-upgrade email: fires when a user hits a real usage quota.
April 2026
6-layer scoring, lifecycle stages, live dashboard
New
  • Launched 6-layer engagement scoring (recency, frequency, depth, key feature, time spent, buying intent) recomputed nightly and live every 3 seconds.
  • Launched 7 automatic lifecycle stages: Signup, Onboarding, Active, Conversion ready, Going quiet, Churned, Paid.
  • Launched the real-time user dashboard with score breakdown, stage badge, and activity feed per user.
  • Added the manual email composer for sending personal messages to any user from the dashboard.
  • Added the tracking snippet API: page views, clicks, time on page, and SPA route changes auto-tracked.
  • Added ccrm.identify() and ccrm.track() client-side methods.
  • Added the REST API for server-side events.
March 2026
Private beta launch
New

ConversionCRM entered private beta with workspace creation, API key provisioning, and the core tracking infrastructure. Beta workspaces lock in their plan price forever.
